# TheReactionTimeBoard
It is a board where you can test your reaction time or battle with someone else to see who has a faster reaction time.

How it works then you press a button, the first red LED turns on. Then the code chooses a random number between 1, 2, or 3, waits that many seconds, and moves on to the next red LED. There are three red LEDs in total. When it reaches the final green LED, a timer starts. When one of the players presses their button, the timer stops. The OLED screen then shows which player won (player 1 or player 2) and what their reaction time was in seconds.

NOTE: The code is in python and if you want to use the exact same code you need to have micropython installed on your esp32
